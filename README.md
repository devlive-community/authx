### 什么是 authx ?
    
---

AuthX 是一个简单、易用的开源权限管理平台。它旨在帮助开发者轻松地实现基于角色的访问控制（RBAC）和权限管理。

### 功能特点

---

- 简单易用的 API 接口
- 支持多种数据库存储
- 可扩展的权限管理策略
- 高性能和高可用性
- 完善的文档和示例
